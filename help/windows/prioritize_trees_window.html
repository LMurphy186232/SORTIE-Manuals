Prioritize Trees To Cut Window

This window is reached from the Edit Harvest Window by clicking on the "Prioritize trees..." button. This is where you enter data on how to prioritize trees to harvest.

You can define up to three priorities.

In the left-most box, select the tree data member to prioritize by. The options in the box depend on the behaviors currently selected for the run. The data member may not apply to all trees being harvested. In this case the priority will be quietly ignored for those trees and they will be treated as non-priority.

Once you have selected the data member, select the range of acceptable values. Values are inclusive. To set only one allowed value, set both the minimum and maximum to this value. Acceptable values for boolean-type data members are 0 for false and 1 for true.
