Model settings window

This window is reached using the menu option "Edit->Model Settings" from the main SORTIE window. It allows you to edit the parameter values for your run.

The options you are given depend on what behaviors have been selected for the model flow. Options for the plot and tree populations are always available.

Each behavior has its own setup button. By default, this opens a window for entering parameter values. For those behaviors that need specialized setup, this button will open the appropriate window.
