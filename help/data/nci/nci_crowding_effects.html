NCI crowding effects

Most crowding effects must be paired with an NCI term. The NCI term returns one or more values representing different indexes of neighborhood competitiveness. Each crowding effect term can be matched with a variety of different competitiveness indexes and each one below describes how it uses each NCI value it receives.

Note: If these effects are used with NCI Mortality, the time period unit for parameters must match the value specified in the NCI Parameter Period (years) parameter.

Crowding no size term

This can be paired with NCI terms returning either one or two values, but it will only use one of them.

Parameters

Parameter nameDescription
NCI Crowding Effect Slope (C) The "C" term in the crowding effect equation.
NCI Crowding Effect Steepness (D) The "D" term in the crowding effect equation.

How it works

Crowding Effect is calculated as:

CE = exp(-C * NCI D)

where:

  • C is the NCI Crowding Effect Slope (C) parameter
  • D is the NCI Crowding Effect Steepness (D) parameter
  • NCI either the second term returned from a multi-value NCI term, or the single value returned from a single-valued NCI

The NCI term is a separate term you select. For a list of options click here. The valid ranges on parameter values will depend on the choice of NCI.
