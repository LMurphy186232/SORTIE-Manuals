Power Function of light

Parameters

NCI Power Shading Effect "a"The "a" parameter in the shading effect equation below. Shape parameter for shade reduction of annual growth.
NCI Power Shading Effect "i"The "i" parameter in the shading effect equation below. Intercept of the light limitation curve.
NCI Power Shading Effect - Max GLI valueThe maximum value that light levels can attain. Used to scale different light indexes to 0-1.

How it works

Shading Effect is calculated as:

SE = (GLI-i)^a

where:

  • a is the NCI Power Shading Effect "a" parameter
  • i is the NCI Power Shading Effect "i" parameter
  • GLI is the light level, from 0-1. If a light index is used that has different units (for instance, Sail Light that goes 0-100), use the NCI Power Shading Effect - Max GLI value to enter the maximum light value and the GLI value will be properly re-scaled from 0-1.

How to apply it

The trees to which this behavior is applied must also have a light behavior applied.
