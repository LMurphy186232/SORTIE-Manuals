Years Since Last Harvest Grid

This grid is created by the Lagged post harvest growth and Post Harvest Skidding Mortality behaviors. Each grid cell holds the time since a harvest last occurred in that cell. The grid cell resolution defaults to 8 m X 8 m. You can set whatever new resolution you wish.

Data in the grid

Data member nameDescription
TimeTime since last harvest
LastUpdatedThe timestep the grid was last updated
