Climate Comp Dep Neighborhood Survival Grid

This grid is created by the Climate and Competition Dependent Neighborhood Survival behavior. It holds the timestep survival rate for each species in each grid cell.

Data in the grid

Data member nameDescription
Survival for Species XThe survival rate for Species X for that cell. The rate is per timestep, not annual. Note that the value may be -1 if there were no trees of Species X in the grid cell for that timestep.
Neighbor basal areaThe adult neighbor basal area, in square meters
