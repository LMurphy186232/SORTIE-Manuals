DetailedSubstrate Grid

This grid is created by the Detailed Substrate behavior. The grid holds the relative proportions of the various substrate types. If the Harvest behavior is present for the run, then this grid's resolution must match the Harvest Results grid. Otherwise, it defaults to a cell resolution of 8 m X 8 m, which you can change. This grid holds packages with a different data structure from the main grid to track Substrate cohorts.
