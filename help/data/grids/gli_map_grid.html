GLI Map [X] Grid

This grid is created by the GLI Map Creator behavior. The grid holds a GLI value in each cell. The grid cell resolution defaults to 8 m X 8 m. You can set whatever new resolution you wish.

Each instance of this grid's name ends with a number, i.e. "GLI Map 3". This is because it is possible to have more than one instance of these in a run. The number corresponds to the position in the behavior list of the parent behavior that created each grid, so they can be distinguished.

Data in the grid

Data member nameDescription
GLIGLI value, as a percentage of full sun between 0 and 100
