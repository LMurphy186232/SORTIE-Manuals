Harvest Results Grid

This grid is created by the Harvest behavior. This is where data on harvest results is stored. The data is stored raw - no conversion to per-hectare amounts.

The grid cell resolution is always set to match the Harvest Master Cuts grid. Any changes you make to the grid cell resolution will be ignored.

Data in the grid

Data member nameDescription
Harvest TypeType of harvest that occurred in the current timestep, or -1 if none has occurred
Cut Density, species x, cut range yNumber of trees cut in the current timestep for the given species and cut range
Cut Basal Area, species x, cut range yTotal basal area cut in the current timestep for the given species and cut range
Cut Seedlings, species xSeedlings killed as part of a harvest event in the current timestep for the given species
