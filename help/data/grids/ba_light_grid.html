Basal Area Light Grid

This grid is created by the Basal Area Light behavior. Each grid cell holds tree basal area data and a light level. The grid cell resolution defaults to 8 m X 8 m. You can set whatever new resolution you wish.

Data in the grid

Data member nameDescription
LightThe light level as calculated by the Basal Area Light behavior
Conifer Basal AreaThe basal area, in square meters, of conifers that count towards the light calculation
Angiosperm Basal AreaThe basal area, in square meters, of angiosperms that count towards the light calculation
