Foliar Chemistry Grid

This grid is created by the Foliar Chemistry behavior. This is where the amount of different foliar chemistry components is stored. All data is stored raw, no conversion to per-hectare amounts.

The grid cell resolution is set to 8 m X 8 m. You can change this to whatever you wish.
