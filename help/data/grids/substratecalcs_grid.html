Substrate calcs Grid

This grid is called "substratecalcs" and is created by the Substrate behavior. This grid is used for intermediate calculations when calculating the values in the Substrate grid. The grid cell resolution must match Substrate's.

Data in the grid

Data member nameDescription
Amount of new tip-up moundsNew tip-up mounds area by grid cell, in square meters
Fresh logs X timesteps agoFresh logs added X timesteps ago up to the value in the Substrate parameter "Maximum number of years that decay occurs" (converted to timesteps), as a proportion of grid cell area
