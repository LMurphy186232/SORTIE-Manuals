Constant GLI

Parameters for this behavior

Parameter nameDescription
Constant GLI - Constant GLI Value (0-100) The GLI value assigned to all trees.

How it works

This assigns a constant GLI value to all trees to which it is assigned. This value is set in the Constant GLI - Constant GLI Value (0-100) parameter.

How to apply it

This behavior may be applied to seedlings, saplings, and adults of any species.
