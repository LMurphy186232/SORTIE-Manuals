How grids are created and used

Which grids are used for a run depends entirely on which behaviors are used. All grids are created automatically by a behavior. A grid that is not needed in a run will not be created.

Behaviors use grids for three primary reasons:

  • To map spatial variation in some aspect of the plot (as with a storm damage susceptibility map)
  • To pass information from behavior to behavior (as with a substrate map which is updated by one behavior and used by another to determine likelihood of seed establishment)
  • To report something for output (number of trees harvested, amount of biomass)
