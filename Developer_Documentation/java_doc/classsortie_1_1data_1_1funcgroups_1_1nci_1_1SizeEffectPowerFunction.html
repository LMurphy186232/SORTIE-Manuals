Detailed Description

This represents the default size effect in NCI growth.

This is not meant to be part of the growth behaviors collection, but to be accessed by NCI.

Size Effect is calculated as:

Size Effect = a * diamb

where:

  • diam is the diameter of the target tree, in cm
  • a and b are parameters
