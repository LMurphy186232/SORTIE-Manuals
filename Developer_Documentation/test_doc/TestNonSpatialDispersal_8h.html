const char* WriteNonSpatialDispersalXMLFile1 ( )

This class tests the non-spatial dispersal behavior.

It writes out parameter files and uses them as input to perform setup; it then sets up the desired initial conditions, causes the behavior to run, then tests its output. There will be two normal processing runs. Both use deterministic seed numbers, which should equal the expected mean +- 1. Writes a parameter file to direct testing using the first set of testing conditions. This contains all tree information and disperse information. It also has a tree map so we'll be dealing with a known list of trees. The timestep length is 1 year.

Returns
Filename written.
